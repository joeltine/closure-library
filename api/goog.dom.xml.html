Creates an XML document appropriate for the current JS runtime

Parameters

The root tag name.

opt_namespaceUristring=

Namespace URI of the document element.

opt_preferActiveXboolean=

Whether to default to ActiveXObject to create Document in IE. Use this if you need xpath support in IE (e.g., selectSingleNode or selectNodes), but be aware that the ActiveXObject does not support various DOM-specific Document methods and attributes.

Returns

The new document.
