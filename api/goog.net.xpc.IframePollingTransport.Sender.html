class Sender

goog.net.xpc.IframePollingTransport.Sender

Utility class to send message-parts to a document from a different origin.

new Sender(url, windowObj)

Parameters

The url the other document will use for polling. Must be an http:// or https:// URL.

send(payload)

Sends a message-part (frame) to the peer. The message-part is encoded and put in the fragment identifier of the URL used for sending (and belongs to the origin/domain of the peer).

Parameters
payloadstring

The message to send.
