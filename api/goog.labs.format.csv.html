parse(text, opt_ignoreErrors, opt_delimiter)

Parses a CSV string to create a two-dimensional array.

This function does not process header lines, etc -- such transformations can be made on the resulting array.

Parameters
textstring

The entire CSV text to be parsed.

opt_ignoreErrorsboolean=

Whether to ignore parsing errors and instead try to recover and keep going.

opt_delimiterstring=

The delimiter to use. Defaults to ','

Returns
!Array<!Array<string>>

The parsed CSV.
