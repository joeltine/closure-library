HttpStatus.isSuccess(status)

Returns whether the given status should be considered successful.

Successful codes are OK (200), CREATED (201), ACCEPTED (202), NO CONTENT (204), PARTIAL CONTENT (206), NOT MODIFIED (304), and IE's no content code (1223).

Parameters
statusnumber

The status code to test.

Returns
boolean

Whether the status code should be considered successful.
