class Ctr

Implementation of Ctr mode for block ciphers. See http://en.wikipedia.org/wiki/Block_cipher_modes_of_operation #Cipher-block_chaining_.28Ctr.29. for an overview, and http://csrc.nist.gov/publications/nistpubs/800-38a/sp800-38a.pdf for the spec.

new Ctr(cipher)

Parameters
cipher!goog.crypt.BlockCipher

The block cipher to use.

Instance Methods

code

decrypt(cipherText, initialVector)

Decrypts a message. In CTR, this is the same as encrypting.

Parameters
cipherText(!Array<number>|!Uint8Array)

Message to decrypt. The length does not have to be a multiple of the blocksize.

initialVector(!Array<number>|!Uint8Array)

Initial vector for the Ctr mode. An array of bytes with the same length as the block size.

Returns
!Array<number>

Decrypted message.

encrypt(plainText, initialVector)

Encrypts a message.

Parameters
plainText(!Array<number>|!Uint8Array)

Message to encrypt. An array of bytes. The length does not have to be a multiple of the blocksize.

initialVector(!Array<number>|!Uint8Array)

Initial vector for the Ctr mode. An array of bytes with the same length as the block size, that should be not reused when using the same key.

Returns
!Array<number>

Encrypted message.
