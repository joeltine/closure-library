on(eventType, callback)

Register a callback to handle I/O events.

See https://iojs.org/api/events.html

Note that under the object mode, an event of DATA will deliver a message of 1) JSON compliant JS object, including arrays; or 2) an ArrayBuffer.

Ordering: messages will be delivered to callbacks in their registration order. There is no ordering between on() and once() callbacks.

Exceptions from callbacks will be caught and ignored.
