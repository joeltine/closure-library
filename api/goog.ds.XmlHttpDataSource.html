class XmlHttpDataSource

goog.ds.DataNode
  └ goog.ds.XmlDataSource
      └ goog.ds.XmlHttpDataSource

Data source whose backing is an XMLHttpRequest,

A URI of an empty string will mean that no request is made and the data source will be a single, empty node.

new XmlHttpDataSource(uri, name)

Parameters
uri(string|goog.Uri)

URL of the XMLHttpRequest.

namestring

Name of the datasource.
