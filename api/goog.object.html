unsafeClone(obj)

Clones a value. The input may be an Object, Array, or basic type. Objects and arrays will be cloned recursively.

WARNINGS: goog.object.unsafeClone does not detect reference loops. Objects that refer to themselves will cause infinite recursion.

goog.object.unsafeClone is unaware of unique identifiers, and copies UIDs created by getUid into cloned results.
