class Cbc

Implements the CBC mode for block ciphers. See http://en.wikipedia.org/wiki/Block_cipher_modes_of_operation #Cipher-block_chaining_.28CBC.29

new Cbc(cipher)

Parameters
cipher!goog.crypt.BlockCipher

The block cipher to use.
